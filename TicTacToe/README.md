# Tic Tac Toe Arena

A simple browser-based Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- Two-player turn-based gameplay (`O` and `X`)
- Live turn and result status
- Win detection with highlighted winning cells
- Scoreboard for `O` wins, `X` wins, and draws
- `Reset Board` and `Reset Score` controls
- Responsive UI for desktop and mobile screens

## Project Structure

- `index.html` - game layout and UI elements
- `style.css` - visual design and responsive styling
- `first.js` - game logic, winner checks, and score handling

## Run Locally

1. Open `index.html` in any modern web browser.
2. Click a cell to place your symbol.
3. Use:
   - `Reset Board` to start a new round
   - `Reset Score` to clear all scores and restart

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
